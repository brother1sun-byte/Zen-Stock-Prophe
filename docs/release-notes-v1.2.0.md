# v1.2.0 個人利用版

## 概要

個人の日常利用を優先し、最初の確認、ウォッチリスト運用、振り返り、データ品質、起動安定性を整理しました。投資助言や注文機能ではなく、一次情報とリスクを確認するための手動判断支援ツールです。

## 主な変更

- 最初に見る画面を「今日の確認」に統一
- 材料、需給、テクニカル、リスク、見送り理由、データ不足を上部へ集約
- タグ、検索、並び替え、「今日確認する銘柄」の端末内保存
- 手動データ編集、キャッシュ確認・削除、API再取得
- After Close Reviewの履歴検索、結果比較、JSON/CSV復元
- 推定、遅延、キャッシュ、補完データ、データ不足の共通表示
- ヘルスチェック、ポート競合回避、ログローテーション、起動待ち
- 390px、430px、768px、デスクトップの回帰QA

## 安全境界

- verified candidateの厳格条件は変更していません。
- 条件未達はresearch-onlyです。
- verified signalがない場合、候補やactionable boardを作りません。
- 自動売買、実注文、証券会社API連携、外部通知、外部ログ送信は行いません。
- 設定、手動データ、レビューはブラウザ内に保存します。

## 既知の制約

- 無料データは遅延、欠損、推定、補完を含みます。
- 実データの取得可否は各API設定と提供状況に依存します。
- 最終判断は利用者が一次情報を確認して手動で行います。
